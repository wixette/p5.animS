p5.animS

p5.animS is a library that animates p5.js shapes by rendering their drawing processes.

Get Started

Download the JavaScript bundle file p5.anims.js from the build dir.

Or, clone the source code and rebuild p5.anims.js:

npm install
npm run build

In your HTML code, put p5.anims.js right after p5.js or p5.min.js:
